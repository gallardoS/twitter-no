# twitter-no

chrome extension that blocks twitter/x with a blank page and a centered `no`.

## features

- on/off toggle from the popup
- closes the twitter/x tab with the `ok` button
- tracks every twitter/x access attempt
- shows total and weekly counts
- weekly heatmap chart
- copies the chart as a png to the clipboard

## install

1. open `chrome://extensions`
2. enable developer mode
3. click `load unpacked`
4. select this folder
