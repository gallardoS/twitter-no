# privacy policy

twitter-no stores only the data needed to work:

- whether the extension is enabled or disabled
- the total number of twitter/x access attempts
- timestamps of twitter/x access attempts for the weekly chart

this data is stored locally in the browser using chrome storage.

twitter-no does not collect personal information, does not use analytics, does not use remote code, and does not sell, share, or transfer user data to third parties.

the extension only runs on `twitter.com` and `x.com`.
